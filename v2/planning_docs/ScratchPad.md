# Scratch Pad

---

## Gameplay ideas

Delivery game play

- start mission to deliver x item in y time
- Doing tricks gains diff XP type (followers/respect), acts as multiplier for $
- XP to unlock bikes
- $ to unlock mods / skins

- Start off on bicycle
- Scooter
- Slow motorcycle
- Stunt motorcycle

- Running from the cops
  - they put up obstacles that you can stunt around

MP

- Friends all work for a company, deliver more dangerous items as you level up for more money
- start by delivering newspaper, then pizza, then weed, then meth, then guns
- More danger = more wanted level

- most deliveries w/ highest score

- Cop chases
